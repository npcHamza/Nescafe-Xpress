# Nescafe XPress

Nescafe XPress, metinleri ikili (binary) formata dönüştürmenizi sağlayan basit bir web uygulamasıdır. Aşağıda bu projenin HTML kodunu inceleyebilirsiniz.

## Kullanılan Teknolojiler

Bu proje aşağıdaki teknolojileri içermektedir:
- HTML
- CSS
- JavaScript
- Bootstrap
- jQuery
- Font Awesome

## Görünüm ve Tarayıcı Uyarlaması

Proje, Nescafe XPress logosu ve bir metin giriş alanı içerir. Kullanıcı metin girdikten sonra "Dönüştür" düğmesine tıkladığında, metin ikili formata dönüştürülür ve altındaki alana görüntülenir. Ayrıca, kullanıcılar sonucu Twitter üzerinde paylaşabilirler.


## Nasıl Çalışır

Kullanıcılar metin girişini "Text to Convert" alanına yapıştırabilir veya yazabilirler. Ardından, "Dönüştür" düğmesine tıkladıklarında JavaScript işlevi çalışır ve metni ikili forma dönüştürür. Sonuç "outputBinary" alanında görüntülenir. Ayrıca, "Tweetle" düğmesine tıklanarak sonucu Twitter üzerinde paylaşabilirsiniz.

Bu proje [npcHamza](https://github.com/npcHamza) tarafından geliştirilmiştir.
